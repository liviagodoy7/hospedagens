O que significa a sigla PHP?

PHP é um acrônimo recursivo para PHP: Hypertext Preprocessor (Pré-Processador de Hipertexto), que originalmente se chamava Personal Home Page (Página Inicial Pessoal). Ele também é um subconjunto de linguagens de scripts como JavaScript e Python. A diferença é que PHP costuma ser mais usado para comunicação do lado do servidor (back-end).

Quais são as principais características da linguagem PHP?

A linguagem PHP é voltada ao desenvolvimento de aplicações Web e, por isso, seu principal objetivo é a implementação veloz, simples e eficiente de soluções para uso em sites. Entre suas principais características estão:

  • Velocidade e robustez no uso.
  • Desenvolvimento orientado a objetos.

Quais funções básicas do PHP você conhece e utiliza em seus projetos?

Algumas das funções mais comuns que utilizo são:

  • mysqli_query($conexao, $consulta) – Executa uma consulta SQL no banco de dados.
  • mysqli_connect($host, $usuario, $senha, $banco) – Estabelece uma conexão com o banco de dados MySQL.
  • if, else, elseif – Estruturas condicionais para tomada de decisão.
  • echo e print – Exibem valores na tela.
  • isset() – Verifica se uma variável está definida e não é null.
  • empty() – Verifica se uma variável está vazia.
  • mysqli_fetch_assoc() – Recupera uma linha do resultado da consulta como um array associativo.

O que significa a sigla XAMPP e qual é a sua principal funcionalidade?

A sigla XAMPP significa X (cross-platform), Apache, MySQL, PHP e Perl. O XAMPP é uma distribuição de código aberto que fornece uma maneira fácil de instalar um servidor local em seu computador para desenvolvimento de aplicações web. Ele combina várias tecnologias essenciais para o desenvolvimento web, facilitando o processo de criação de ambientes de desenvolvimento sem a necessidade de configurar cada componente individualmente.

A principal funcionalidade do XAMPP é permitir que os desenvolvedores criem um ambiente de servidor local em suas máquinas de forma rápida e fácil. Ele proporciona tudo o que é necessário para desenvolver e testar aplicações web que dependem do Apache, MySQL e PHP (ou Perl), sem a necessidade de configurar manualmente cada componente.

Quais componentes principais fazem parte do XAMPP? Explique a função de cada um

Os principais componentes são:

  • Apache: Processa requisições HTTP e fornece páginas web.
  • MySQL / MariaDB: Sistema de gerenciamento de banco de dados relacional (RDBMS).
  • PHP: Linguagem de programação usada para criar conteúdo dinâmico para páginas web.
  • Perl: Linguagem usada para desenvolvimento web e manipulação de texto.
  • PhpMyAdmin: Interface gráfica para gerenciar bancos de dados MySQL/MariaDB.

O que é uma tabela no MySQL? Explique o seu conceito e a sua importância dentro de um banco de dados.

Uma tabela no MySQL é uma estrutura que organiza dados em linhas e colunas. Cada coluna representa um tipo específico de dado (como texto, números ou datas), e cada linha contém um conjunto de valores que correspondem às colunas da tabela. Por exemplo, em uma tabela de clientes, as colunas podem ser nome, e-mail e idade, enquanto cada linha seria um registro de cliente com seus dados. A tabela é essencial porque permite o armazenamento estruturado e eficiente das informações, além de facilitar a consulta e manipulação dos dados com comandos SQL. Ela também permite o relacionamento entre diferentes conjuntos de dados, como conectar informações de clientes com seus pedidos por meio de uma chave estrangeira.

Quais são os principais tipos de dados utilizados em tabelas MySQL?

No MySQL, os tipos de dados são usados para definir a natureza das informações que serão armazenadas em cada coluna de uma tabela. Alguns dos principais tipos de dados utilizados em tabelas MySQL são:

  • INT: Utilizado para armazenar números inteiros, como idades ou quantidades.
  • VARCHAR: Usado para armazenar strings de texto de comprimento variável. Ideal para armazenar nomes, e-mails, endereços, etc.
  • TEXT: Semelhante ao VARCHAR, mas usado para armazenar textos mais longos, como descrições ou comentários.
  • DATE: Armazena datas no formato 'YYYY-MM-DD', como datas de nascimento ou de criação de registros.
  • DATETIME: Armazena data e hora, no formato 'YYYY-MM-DD HH:MM:SS', usado quando é necessário registrar informações com precisão de data e hora.
  • DECIMAL: Usado para armazenar números decimais com precisão exata, como valores monetários.
  • FLOAT e DOUBLE: Usados para armazenar números de ponto flutuante.
  • BOOLEAN: Armazena valores booleanos, representando verdadeiro ou falso (1 ou 0).

O que é uma chave primária (Primary Key) em uma tabela do MySQL? Qual a sua importância?

Uma chave primária (Primary Key) em uma tabela MySQL é uma coluna (ou um conjunto de colunas) que serve para identificar de forma única cada registro dentro da tabela. Ela garante que cada linha na tabela tenha um valor exclusivo e não nulo, o que impede que existam registros duplicados.

A importância da chave primária está na sua capacidade de garantir a unicidade dos dados e ajudar na eficiência das consultas. Ela facilita a indexação, o que melhora a performance ao buscar ou alterar registros. Além disso, a chave primária é essencial para estabelecer relacionamentos entre tabelas, já que outras tabelas podem referenciar a chave primária através de uma chave estrangeira (Foreign Key), criando vínculos entre dados em diferentes tabelas.

Como você faria a conexão com um banco de dados MySQL utilizando PHP?

A conexão com o banco de dados MySQL pode ser feita utilizando a função mysqli_connect(), como segue:


$conexao = mysqli_connect("localhost", "usuario", "senha", "banco_de_dados");
if (!$conexao) {
    die("Falha na conexão: " . mysqli_connect_error());
}
